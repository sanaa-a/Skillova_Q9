# Skillova Q9

A single-page portfolio landing page built with **Next.js 16** (App Router, TypeScript, Tailwind CSS v4). Project case-study copy is fetched from **DummyJSON's `/posts` endpoint** at build time paired with real `reactions`/`views` numbers shown on each card. The page ships as static HTML with no client-side loading spinners.

## Run it locally
1. download repo then
2. extraxt all
3. open in visual studio
4. then run
5. npm install
6. npm run dev

Open http://localhost:3000.
